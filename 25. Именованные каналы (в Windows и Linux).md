## Теория 
**Именованные каналы (в Windows и Linux). Назначение и использование**

Именованный канал — это двунаправленный или однонаправленный канал связи, созданный в файловой системе. Он представляет собой специальный файл, который позволяет различным процессам обмениваться данными независимо от родственных связей. В отличие от анонимных каналов, именованные каналы доступны любым процессам, имеющим доступ к файловой системе.

Назначение именованных каналов:
1. Обеспечивают коммуникацию между неродственными процессами.
2. Являются удобным механизмом IPC (Inter-Process Communication — межпроцессного взаимодействия).
3. Позволяют отправлять и получать данные между любыми процессами, даже если они не связаны семейными отношениями (parent-child).

Основные особенности:
1. Название: Канал имеет имя, видимое в файловой системе, что позволяет другим процессам находить и подключаться к каналу.
2. Двунаправленность: Некоторые именованные каналы допускают двустороннюю связь (например, в Windows), другие — одностороннюю (например, в Linux).
3. Постоянность: Канал существует в файловой системе до момента удаления.

## Практика 

Создание линейного односвязного списка из n элементов. Значение элемента генерируется случайным образом. Поместить в отдельные указатели адрес минимального и максимального элемента списка. Вывести список и значения минимального и максимального элемента через указатель (отдельно для Windows и Linux).

``` C
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <stdlib.h>
#include <malloc.h>
#include <time.h>

typedef struct {
	int value;
	struct ListEl* next;
} ListEl;

ListEl* CreateList(int count)
{
	int min = 10;
	int max = 100;
	srand(time(0));
	ListEl* FirstEl = malloc(sizeof(ListEl));
	FirstEl->value = min + rand() % (max - min);
	ListEl* PostEl, *NextEl;
	PostEl = FirstEl;
	for (int i = 0; i < count - 1; i++)
	{
		NextEl = malloc(sizeof(ListEl));
		NextEl->value = min + rand() % (max - min);
		PostEl->next = NextEl;
		PostEl = NextEl;
	}
	PostEl->next = NULL;
	return FirstEl;
}

ListEl* SearcMaxEl(ListEl* list)
{
	ListEl* max = list;
	while (list)
	{
		if (list->value > max->value) max = list;
		list = list->next;
	}
	return max;
}

ListEl* SearcMinEl(ListEl* list)
{
	ListEl* min = list;
	while (list)
	{
		if (list->value < min->value) min = list;
		list = list->next;
	}
	return min;
}

void PrintAllList(ListEl* list) {
	int index = 1;
	while (list) {
		printf("Элемент %d = %d\n", index, list->value);
		index++;
		list = list->next;
	}
}

int main() {
	system("chcp 1251");
	ListEl* list = CreateList(15);
	ListEl* max = SearcMaxEl(list);
	ListEl* min = SearcMinEl(list);
	PrintAllList(list);
	printf("max = %d\n", max->value);
	printf("min = %d\n", min->value);
}
```
