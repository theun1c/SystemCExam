## Теория 

Системное программирование — это направление разработки программного обеспечения, направленное на непосредственное взаимодействие с аппаратным уровнем компьютера и операционной системой. Цель системного программирования — создание специализированных программ, управляющих устройствами, организующими хранение данных и поддерживающих функционирование оборудования на самом нижнем уровне.

Примеры:

1.       драйвер мыши

2.       файловая система

3.       служба Windows

Специфика системного программирования:
1.       Прямая работа с железом: Управляешь самим оборудованием компьютера (видеокартами, сетевыми картами, жёсткими дисками и другими устройствами).
2.       Распределение ресурсов: Контролируешь память, мощность процессора и передачу данных.
3.       Большая ответственность: Ошибки тут серьёзно влияют на работу всей системы — вплоть до её зависания или поломки.
4.       Минимум абстракций: Близкое общение с внутренностями компьютера, никаких лишней прослоек и вспомогательных программ.
5.       Особые знания: Нужно глубоко разбираться в устройстве компьютера и особенностях операционных систем.
6.       Максимальная производительность: Твой код должен быть предельно быстрым и экономичным по ресурсам.
## Практика 
Программа для преобразования строки в число. Строка хранится в виде указателя LPWSTR. Реализовать без использования системных функций

Шаг 1. Для этого задания в файле main.c напишем следующий код

``` C
#include <Windows.h>
#include <stdio.h>

// преобразование строки WINAPI в число
int LPWSTR_to_int(LPWSTR str) {
	int result = 0;
	int sign = 1;
	int i = 0;

	// убрал все пробелы в самомначале
	while (str[i] == L' ') {
		i++;
	}

	// если есть - то число отрицательное
	if (str[i] == L'-') {
		sign = -1;
		i++;
	}

	// пока у нас числа мы конвертируем
	while (str[i] >= L'0' && str[i] <= L'9') {
		result = result * 10 + (str[i] - L'0');
		i++;
	}

	// возвращаем число + меняем знак
	return result * sign;
}

int main() {
	system("chcp 1251");
	LPWSTR str = L"123";

	int num = LPWSTR_to_int(str);
	printf("%d", num);
}
```
