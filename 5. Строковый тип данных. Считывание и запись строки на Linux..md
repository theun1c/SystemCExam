Строки в C:

Строка в языке C — это последовательность символов, заканчивающаяся специальным символом конца строки — '\0' (нулевой байт). Она хранится в виде массива типа char. Именно наличие символа окончания отличает строку от обычного массива символов фиксированной длины.

Ключевые моменты при работе со строками в C:

1.       Невозможно изменять длину строки простым присваиванием нового значения. Для изменения строки надо сначала выделять новую память и копировать туда символы.

2.       Легко выйти за пределы массива. При попытке записать символы за пределами выделенного массива произойдет ошибка сегментации или повреждение данных.

3.       Операции со строками выполняются через библиотечные функции. Такие функции, как strcpy, strlen, strcmp, являются основными инструментами для работы со строками.

4.       Необходимо аккуратно обращаться с памятью. При создании строк динамического размера обязательно следите за выделением и освобождением памяти, иначе возникнут утечки памяти или аварийные завершения программы.

``` C
#include <unistd.h>
#include <stdio.h>
#include <string.h>

int main(){
	int file = open("file.txt", O_RDWR | O_CREAT, S_IRWXU);
	char* buffer = calloc(100, sizeof(char));
	
	int readIs = read(file, buffer, 100);
	
	if(readIs == -1){
		buffer[strlen(buffer)] = '\0';
		
		int fileWrite = open("answer.txt", O_RDWR | O_CREAT | O_TRUNC, S_IRWXU);
		
		int isWrite = write(fileWrite, buffer, strlen(buffer));
	}
}

```
