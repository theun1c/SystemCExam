## Билет 17

**Указатель как тип данных. Особенности работы с указателями**

**Указатели** - это переменные, которые хранят в памяти адрес другой переменной. Вместо того, чтобы содержать фактическое значение, указатель содержит местоположение (адрес), в котором значение хранится в памяти компьютера.

Пример:

![Photo](assets/Pasted%20image%2020251226054900.png)

 Особенности работы с указателями:
1. Аккуратность с памятью: Указатели дают прямой доступ к ячейкам памяти, что может привести к ошибкам, если работать небрежно. Важно точно указывать, куда ссылается указатель, и не вызывать указатели на несуществующие или неправильные адреса.
2. Инициализация указателей: Никогда не используйте неинициализированные указатели. Обращение к указателю, который не указывает ни на какую действительную область памяти, приводит к неопределенному поведению или аварийному завершению программы.
3. Разыменование указателя: Чтобы получить значение, на которое указывает указатель, используется оператор разыменования
4. Оператор адреса (&): Чтобы получить адрес переменной, используется оператор &.
5. Указатели на указатели: Возможно создавать указатели на указатели, что добавляет дополнительный уровень косвенности.

Применение указателей:
1. Работа с динамической памятью: Для выделения и освобождения памяти используются операторы malloc/free или new/delete, возвращающие указатели на выделенные блоки памяти.
2. Параметры функций: Указатели часто передают в функции, чтобы менять значения переменных снаружи.
3. Работа с массивами: Массивы фактически реализованы как указатели, что позволяет эффективно перемещаться по элементам массива с помощью арифметики указателей.

## Практика 
Запись и считывание числовых параметров в системный реестр

Шаг 1. Для выполнения задания необходимо написать следующий код в main.c

``` C
#include <Windows.h>
#include <stdio.h>

int main() {
	system("chcp 1251");

	HKEY hKey;
	if (RegOpenKeyA(HKEY_CURRENT_USER, 0, &hKey) == 0) {
		DWORD n = 30;
		if (RegSetValueExA(hKey, 0, 0, REG_DWORD, &n, sizeof(DWORD)) == 0) {
			DWORD type;
			DWORD data;
			DWORD len;
			if (RegGetValueA(hKey, 0, 0, RRF_RT_REG_DWORD, &type, &data, &len) == 0) {
				printf("%d\n", data);
			}
		}
	}
}
```
