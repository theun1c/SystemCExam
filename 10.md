# 10 Билет
## Теория

Препроцессор - это специальная программа, которая осуществляет алгоритмические действия перед компиляцией основного кода.

Команды для препроцессора называются директивами.

Файл заголовка является текстовым файлом, который подключается к основным файлам проекта с помощью директивы #include

Заголовочные файлы создаются с целью размещения вспомогательной информации, которая используется для работы программы.

Заголовочные файлы бывают системные (подключаются с использованием угловых скобок. Пример: #include.h) и пользовательские (подключаются с использованием кавычек. Пример: #include "myheader.h")

Предназначение

Директивы используются для обработки условий, которые нужно обработать до компиляции основного кода, для создания констант, для определения макросов

Заголовочные файлы разделение кода на логические части, сокращения дублирования кода, улучшение читаемости скрытие деталей реализации

Особенности включенный код предотвращения двойных включений Использование в разных файлах переносимость

## Практика

``` C

#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <Windows.h>

void Set(LPWSTR str) {
	int len = strlen(str) * sizeof(LPWSTR);
	HANDLE h = GlobalAlloc(0, len);
	memcpy(GlobalLock(h), str, len);
	GlobalUnlock(h);
	OpenClipboard(0);
	EmptyClipboard();
	SetClipboardData(CF_UNICODETEXT, h);
	CloseClipboard();
}

int Get() {
	OpenClipboard(0);
	MessageBox(0, GetClipboardData(CF_UNICODETEXT), L"Заголовок", 0);
	CloseClipboard();
}

int main() {
	Set(L"Я умру скоро....");
	Get();
}
```
